CyberOne 软件发布说明
=====================

将你的软件文件放在这个文件夹中，然后使用以下命令发布：

gh release create v1.0.0 releases/你的软件.exe --title "CyberOne v1.0.0" --notes "版本说明"

示例：
gh release create v1.0.0 releases/CyberOne.exe --title "CyberOne安全工具 v1.0.0" --notes "首次发布版本"

发布后，下载链接格式：
https://github.com/lik699386-ctrl/cyberone-website/releases/download/v1.0.0/CyberOne.exe

中国大陆加速下载（使用镜像）：
https://ghproxy.com/https://github.com/lik699386-ctrl/cyberone-website/releases/download/v1.0.0/CyberOne.exe
