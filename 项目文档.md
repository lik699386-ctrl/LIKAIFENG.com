## 项目概述

**LIKAIFENG.Net** 是一个现代化的个人网站项目，采用最新的Web设计标准和Apple风格的交互体验。

### 核心信息

- **项目名称**: LIKAIFENG.Net
- **当前版本**: v4.1.2 (2025-10-17)
- **在线地址**: [https://lik699386-ctrl.github.io/likaifeng.com/](https://lik699386-ctrl.github.io/likaifeng.com/)
- **主要技术**: HTML5 + CSS3 + Vanilla JavaScript
- **架构特点**: 单文件架构，零外部依赖

### 设计理念

- **Apple风格交互** - iOS级别的丝滑动画和交互体验
- **玻璃态设计** - 现代化的半透明效果和背景模糊
- **多语言支持** - 完整的6语言国际化系统
- **响应式设计** - 完美适配桌面、平板、手机
- **无障碍优先** - ARIA支持，键盘导航，动画偏好检测

## 核心特性

### ✨ 多语言国际化
- **6种语言完整支持**：简体中文（默认）、English、Русский、Français、Deutsch、Español

### 支持的语言

| 语言 | 代码 | 覆盖率 | 状态 |
|------|------|--------|------|
| 简体中文 | zh-CN | 100% | ✅ 默认 |
| English | en | 100% | ✅ 完整 |

| Français | fr | 100% | ✅ 完整 |
| Deutsch | de | 100% | ✅ 完整 |
| Español | es | 100% | ✅ 完整 |

### 翻译架构

基于`data-i18n`属性的精确翻译系统：

```html
<!-- HTML元素标记 -->
<h1 data-i18n="hero.title.line1">为你打造的艺术品</h1>
<button data-i18n="hero.cta">浏览</button>
```

```javascript
// 翻译字典结构
const translations = {
    'zh-CN': {
        'hero.title.line1': '为你打造的艺术品',
        'hero.cta': '浏览'
    },
    'en': {
        'hero.title.line1': 'Artwork made especially for you',
        'hero.cta': 'View'
    }
};
```

### 语言切换逻辑

1. **用户选择语言** → 点击下拉菜单中的语言选项
2. **更新localStorage** → 保存用户偏好
3. **动态更新DOM** → 遍历所有`[data-i18n]`元素
4. **跨页面同步** → 其他页面自动应用相同语言

## 更新日志

### v4.1.2 (2025-10-17) - 当前版本
**🐛 Bug修复版本**

#### Bug修复
- 🐛 **修复Umail产品链接点击无响应问题（电脑端）**
  - **问题原因**：下拉菜单点击事件处理使用`e.target.tagName === 'A'`检测链接，当点击链接内文本时无法正确识别
  - **修复方案**：改用`e.target.closest('a')`查找最近的父级链接元素，确保无论点击链接还是链接内文本都能正常工作
  - **影响范围**：桌面端产品下拉菜单中的所有链接（主要是Umail链接）
  - **技术细节**：
    - 在语言选择事件处理中添加链接检测，避免干扰产品链接点击
    - 优化链接点击时的菜单关闭逻辑，确保跳转前先关闭菜单
    - 排除锚点链接（`#`开头）的特殊处理

#### 技术改进
- ✨ **优化下拉菜单链接交互体验**
  - 点击有效链接时自动关闭菜单并完成导航
  - 改进了事件冒泡处理，避免多个事件处理器互相干扰
  - 增强了链接元素检测的健壮性
